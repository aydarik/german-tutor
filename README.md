# German Tutor 🇩🇪

A simple Flashcards web application designed to help learn German vocabulary directly from Markdown notes.

## 📝 Markdown Format

**Add your vocabulary**:
Place your Markdown files (e.g., `Vokabeln.md`, `Wortschatz.md`) in the `work/` directory.

The parser expects vocabulary entries in a format similar to:
```
- [ ] **der Besitzer, - / die Besitzerin, -nen** (владелец, владелица / owner) - Jemand, dem etwas gehört (z.B. eine Firma oder ein Haus).
	- *Beispiel:* Der Besitzer des Cafés ist sehr freundlich.

```
